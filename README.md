# COVID-19 dashboard


## Bokeh app

* `corona-app`
    * `tabs` : scripts defining tabs (`draw_map.py` and `draw_timeseries.py`)
    * `main.py` : main script
    * `utils.py` : collection of functions
    * `theme.yaml` : theme settings
* to run: `bokeh serve --show .` inside `corona-app` directory
